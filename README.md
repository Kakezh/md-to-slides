# MD-to-Slides

将 Markdown 大纲文档转换为精美的 HTML 幻灯片演示文稿。

## 功能特性

- **Markdown 驱动** - 只需提供结构化的 Markdown 大纲，自动生成完整幻灯片
- **多主题支持** - 内置赛博朋克、极简商务、暗夜学术、科技未来四种视觉风格
- **16:9 画幅** - 严格适配标准演示比例，无滚动条设计
- **丰富动效** - 入场动画、霓虹发光、数据计数器等视觉效果
- **数据可视化** - 自动识别数值数据，生成 Chart.js 图表
- **键盘导航** - 支持方向键、空格键、全屏快捷键
- **触屏支持** - 左右滑动翻页，适配移动设备
- **iframe 架构** - 每页独立 HTML 文件，便于维护和扩展

## 快速开始

### 基础用法

```
帮我把这份 markdown 大纲做成 HTML 幻灯片，风格要赛博朋克
[附带 .md 文件]
```

### 完整用法（推荐）

```
帮我从 md 文档出发制作一个用于演示的幻灯片材料，具体要求：
1. 深入研读大纲，合理规划每页内容和布局
2. html 展示文件以 iframe 静态页面模式构建
3. 统一风格 CSS，主题色配合内容选取
4. 引入 js 动效，数据部分用图表展示
5. 标题/金句/人名/洞察要有层级区分
6. 16:9 画幅，无滚动条
7. 键盘翻页 + 全屏快捷键
8. 整体风格为 [赛博朋克 / 极简商务 / 暗夜学术 / 科技未来]
```

## 输入格式

用户提供类似以下结构的 Markdown 文件：

```markdown
# 幻灯片主题标题

## 幻灯片 1：封面页
- **标题**：主标题文字
- **副标题**：副标题文字

## 幻灯片 2：[节名]
- **标题**：本节标题
- **核心问题/观点**：正文内容
- **数据**：数值对比数据（会自动生成图表）

## 幻灯片 3：[节名]
### 子话题 A
- 要点一
- 要点二

## 幻灯片 N：总结
- **金句**：> 引用格式的金句会被特殊展示
```

### 格式约定

| 语法 | 效果 |
|------|------|
| `## 幻灯片 N` | 对应一页幻灯片 |
| `**加粗**` | 关键词高亮展示 |
| `> 引用` | 金句特殊样式 |
| 数值对比 | 自动生成图表 |
| 人名 | 特殊标注展示 |

## 主题风格

| 主题 | 背景 | 主色 | 适用场景 |
|------|------|------|----------|
| **赛博朋克** | `#050510` 深空 | `#00f5ff` 霓虹青 | 科技、创新、前沿话题 |
| **极简商务** | `#ffffff` 白 | `#1a1a2e` 深蓝 | 商务汇报、正式场合 |
| **暗夜学术** | `#1a1a2e` 深蓝 | `#e8e8e8` 浅灰 | 学术报告、研究分享 |
| **科技未来** | `#0c0c1d` 深紫 | `#00d4ff` 电蓝 | 产品发布、技术演示 |

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| `→` / `↓` / `Space` / `PageDown` | 下一页 |
| `←` / `↑` / `PageUp` | 上一页 |
| `Home` | 跳到首页 |
| `End` | 跳到末页 |
| `F` | 切换全屏 |
| `Escape` | 退出全屏 |

## 本地运行

直接在浏览器中打开 `index.html` 即可运行。

如果遇到跨域限制（部分浏览器对本地 iframe 有限制），可通过本地服务器运行：

```bash
# 方式 1：VS Code Live Server 插件

# 方式 2：npx serve
npx serve ./project-root

# 方式 3：Python
python -m http.server 8080 --directory ./project-root
```

## 技术栈

- **HTML5** - 语义化结构
- **CSS3** - 动画、渐变、网格布局
- **JavaScript** - 导航、动效、交互
- **Chart.js** - 数据可视化（CDN）
- **Google Fonts** - Orbitron、Noto Sans SC 等
