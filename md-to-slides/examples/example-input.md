# 示例输入：用户提供的 Markdown 大纲格式

用户应提供类似以下结构的 Markdown 文件：

---

```markdown
# 幻灯片主题标题

## 幻灯片 1：封面页
- **标题**：主标题文字
- **副标题**：副标题文字

## 幻灯片 2：[节名]
- **标题**：本节标题
- **核心问题/观点**：正文内容
- **数据**：数值对比数据（会自动生成图表）

## 幻灯片 3：[节名]
### 子话题 A
- 要点一
- 要点二

### 子话题 B
- 要点一
- 要点二

## 幻灯片 N：总结
- **金句**：> 引用格式的金句会被特殊展示
```

---

## 关键约定
1. 每个 `## 幻灯片 N` 对应一页（内容过多时 Agent 会自动拆分）
2. `**加粗**` 标记的关键词会被高亮展示
3. `> 引用` 格式的文字会被作为金句处理
4. 包含数字对比的内容会自动生成图表
5. 人名会被特殊标注展示
